# 字幕时间轴播放器使用流程闭环指南

## 你的当前使用场景

你会在一个目录里准备好：

- `xxx.mp4`
- `xxx.md`

它们同名，并且 `xxx.md` 中包含 `## Segments`。

你想要的闭环是：

1. 准备视频和字幕 Markdown
2. 必要时把 Markdown 转成 `.vtt`
3. 启动播放器工程
4. 选择本地目录
5. 在左侧视频下方点击视频列表
6. 自动加载对应视频和字幕

## 推荐闭环流程

### 第一步：准备目录

例如目录 `E:\myvideo\demo-folder` 中有：

- `demo-01.mp4`
- `demo-01.md`
- `demo-02.mp4`
- `demo-02.md`

要求：

- 视频和 Markdown 同名
- Markdown 中有 `## Segments`

### 第二步：如果需要，先把 Markdown 转成 VTT

将任意一个 Markdown 转成 VTT：

```powershell
node E:\myvideo\subtitle-player\convert-md-to-vtt.mjs "E:\myvideo\demo-folder\demo-01.md"
```

默认输出：

- `E:\myvideo\demo-folder\demo-01.vtt`

如果你想指定输出路径：

```powershell
node E:\myvideo\subtitle-player\convert-md-to-vtt.mjs `
  "E:\myvideo\demo-folder\demo-01.md" `
  "E:\myvideo\demo-folder\exports\demo-01.vtt"
```

### 第三步：启动播放器工程

```powershell
powershell -ExecutionPolicy Bypass -File E:\myvideo\subtitle-player\start.ps1
```

如果 `8765` 被占用：

```powershell
powershell -ExecutionPolicy Bypass -File E:\myvideo\subtitle-player\start.ps1 -Port 8766
```

### 第四步：在页面中选择本地文件夹

打开页面后：

1. 点击 `选择本地文件夹`
2. 选中你的目录，例如 `E:\myvideo\demo-folder`
3. 工程会自动扫描目录中的同名 `.mp4 + .md` 配对

### 第五步：点击左侧视频下方的视频列表

扫描成功后：

- 左侧视频下方会出现视频列表
- 每一项对应一组同名视频和 Markdown

点击某一项后会自动完成：

- 加载该视频
- 解析对应 Markdown 的 `## Segments`
- 渲染右侧字幕时间轴
- 生成当前项目的 VTT

### 第六步：按需要打开或关闭视频内字幕

当前默认行为：

- 视频内字幕默认不显示

如果你想显示视频内字幕：

- 勾选页面上的 `显示视频字幕`

如果你只想用右侧字幕列表联动视频：

- 保持该开关关闭即可

### 第七步：在右侧时间轴中使用

- 点击任意字幕，视频会跳转到对应时间
- `[` 跳上一条
- `]` 跳下一条
- 可以在右上方搜索字幕内容
- 可以下载当前自动生成的 `.vtt`

## 如果你不想选目录，只想打开默认项目

直接运行：

```powershell
powershell -ExecutionPolicy Bypass -File E:\myvideo\subtitle-player\start.ps1
```

页面会读取：

- [project.config.json](/E:/myvideo/subtitle-player/project.config.json)

指定的默认视频和 Markdown。

## 如果你想提前把某个项目写成默认项目

```powershell
powershell -ExecutionPolicy Bypass -File E:\myvideo\subtitle-player\start.ps1 `
  -Title "demo" `
  -VideoPath "E:\myvideo\demo-folder\demo-01.mp4" `
  -MarkdownPath "E:\myvideo\demo-folder\demo-01.md"
```

这个命令会：

1. 更新默认配置
2. 导出默认 `.vtt`
3. 启动服务并打开页面

## 以后你最常用的两条命令

任意 Markdown 转 VTT：

```powershell
node E:\myvideo\subtitle-player\convert-md-to-vtt.mjs "E:\path\demo.md"
```

启动播放器：

```powershell
powershell -ExecutionPolicy Bypass -File E:\myvideo\subtitle-player\start.ps1
```

## 最终闭环总结

你之后的完整流程可以固定成：

1. 准备一个目录
2. 把同名 `.mp4` 和 `.md` 放进去
3. 启动播放器工程
4. 在页面中选择这个目录
5. 在左侧下方视频列表中点选目标视频
6. 右侧字幕列表自动联动
7. 需要时再打开视频内字幕
